mepTracker¶

mepTracker is a package for tracking and finding optimal Motor-Evoked Potential (MEPs). By starting with an initial guess, it seeks to determine a path to the region of maximum MEP by reading the coil responses along an optimized path in a 2D or 3D space.

Currently an interface to the Simplex vertex-edge optimization method (sve) is av available. The sve package is required. See the SVE page for more information.

  • Source: https://github.com/yelrew/meptracker
  • Documentation: https://yelrew.github.io/m_e_p_t_r_a_c_k_e_r

Install¶

Install the latest version of mepTracker

$ pip install --user -U git+ssh://git@github.com/yelrew/meptracker

or

$ git clone https://github.com/yelrew/meptracker.git && cd meptracker && pip install -e .
